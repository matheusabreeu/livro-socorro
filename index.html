📖 Como utilizar o sistema:

  • 📝
    Preencha os dados: Tudo fica salvo no seu celular automaticamente. Se a internet cair ou você precisar fechar o site para atender uma ocorrência, não se preocupe, os dados continuam aqui quando voltar.
  • 📋
    Clique no botão verde Copiar Tudo para o SEI: Ao terminar de preencher seu plantão, clique nele. O sistema organizará todas as tabelas e deixará o texto invisível copiado no seu celular.
  • 🌐
    Clique no botão azul Acessar Sistema SEI: Ele vai abrir o site do SEI em outra aba. Faça o login, crie um documento em branco no processo e simplesmente cole (opção colar do celular ou Ctrl+V). As tabelas vão aparecer prontas.
  • ⚠️
    Clique no botão vermelho Apagar Tudo (Novo Plantão): Use este botão apenas na hora que passar o serviço e quiser limpar a tela para a próxima guarnição começar do zero.
